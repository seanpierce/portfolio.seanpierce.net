# portfolio.seanpierce.net

A simple portfolio website for Sean Pierce Sumler (that's me).

## Keeping it simple

This site uses plain html, css, and js. No frameworks, no bloat, no dependencies. You wouldn't use a chainsaw to cut a slice of bread.

## DevOps and hosting

This site is hosted on an Ubuntu cloud server and served using nginx. A Github Action is responsible for automated deployments.
